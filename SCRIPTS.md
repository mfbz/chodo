# Available Scripts

All scripts is available to see in package.json

## Devnet

### Install devnet

```npm
npm run localnet:update
```

### Start devnet

```npm
npm run localnet:up
```

### Stream logs

```npm
npm run localnet:logs
```

### Stop devnet

```npm
npm run localnet:down
```

## Client application

### Start client

```npm
npm start
```

## On-chain program

### Build program

```npm
npm run build
```

### Deploy program to devnet

```npm
npm run deploy
```

### Clean store

```npm
npm run clean
```
